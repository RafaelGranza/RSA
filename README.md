# RSA
C++ RSA Generator and Breaker 

#### To Compile:
```c++
$ make
```

## Encryption Tests

* The text to encrypt is in "*messages/text.txt*".
* The **encrypted** file is in "*messages/encoded.txt*".
* The **decrypted** file is in "*messages/decoded.txt*".

##### To run the test:
```c++
$ ./main
```

## Key Breaker Tests

##### To run the test:
```c++
$ ./break
```
